* 德沃雙拼 / double_pinyin_dvorak
为德沃夏克鍵盤佈局設計的雙拼方案。
** 安裝 / Install
本方案爲 Rime 輸入法引擎的輸入方案。使用前請先保證已經正確安裝 Rime 輸入法引擎。
將 ~double_pinyin_dvorak.schema.yaml~ 複製到 ~Rime~ 目錄下。
將下列代碼添加至 ~Rime/default.custom.yaml~ 中：
#+BEGIN_SRC yaml
patch:
  schema_list:
    - schema: double_pinyin_dvorak
#+END_SRC
重新部署 Rime，即可選擇使用德沃雙拼。
